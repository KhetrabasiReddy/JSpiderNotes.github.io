Database

Database

Database is a software which is used to store the application data.

Oracle DB

It is a RDBMS(Relational Database Management System).

In al RDBMS data will be stored in rows and columns format (or tabular format.)

Student

SId SName SPhno
1 Khetrabasi 199
2 Ram 108
3 Laxman 100

Example of RDBMS: MySQL, Oracle, Postgre SQL, DB2, SQL server

MongoDB

It is a No-SQL DBMS.

In No-SQL DBMS data will be organized in non-tabular format(i.e keys and values pair).

Ex: Student = {SId: 1, SName: "Ram", SPhno:108}

Where SId, SName & SPhno are keys & 1, "Ram" & 108 are values.

SQL( Structured Query Language)

SQL is a collection of commands, clauses, Operators & functions.

It is used to communicate with DB in the form of queries.

Classification of SQL Commands:

  1. Data Query Language(DQL)
  2. Data Defination Language
  3. Data Manipulation Language(DML)
  4. Data Transaction / Transaction Control Language
  5. Data Control Language

Data Query Language(DQL)

To learn DQL commands we use scott user DB.(Scott is a default user in oracle database.)
